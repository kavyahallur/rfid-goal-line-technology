# RFID-Based Goal Line Technology (Arduino)

Embedded Systems Project | Arduino | RFID | Real-Time Detection

## Overview
Designed and implemented an Arduino-based RFID system for real-time goal detection in football, eliminating reliance on traditional camera-based analysis. The system enables instant detection and alert generation when the ball crosses the goal line.

---

## System Architecture & Workflow
1. RFID tag embedded in the football  
2. RFID reader placed near the goal line  
3. Arduino processes incoming RFID signals  
4. System determines goal-line crossing event  
5. Instant alert triggered (LED/Buzzer/Display)  

---

## Key Features
- Real-time goal detection using RFID technology  
- Integration of RFID reader and Arduino firmware  
- Instant signal processing and alert generation  
- Reduced dependency on camera-based systems  

---

## Hardware Components
- Arduino (Uno/Nano)  
- RFID Reader Module (RC522 or similar)  
- RFID Tags  
- Buzzer / LED for alerts  
- Power Supply  

---

## Design Approach
The system was designed to continuously monitor RFID signals near the goal line and trigger an event when a tagged object (football) crosses the detection zone. A lightweight firmware logic ensures low latency and real-time responsiveness.

- Embedded firmware development using Arduino
